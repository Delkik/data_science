Vision Zero's Zero Vision Plan

Abstract

The focus of the project is to provide visual analysis on the various aspects of Vision Zero and other crash data in NYC. This is done through the merging of datasets and the creation of visual tools through matplotlib and seaborn which use the data provided by analysis done on subway, crash, speed limit, and neighborhood slow zone data.
