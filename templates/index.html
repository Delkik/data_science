Overview

Bill DeBlasio's Vision Zero plan did not end up reaching its expected goals. While the plan touted ideas that should have worked in theory, it still could not do enough to reach the goals. I wanted to look at some of these plans to see if they made enough of an impact. I believed that slow zones would have a larger decrease in crashes and that more transportation usage would lead to less crashes. I tested this using NYC Open Data and publicly available MTA turnstile data and visualized the data using seaborn and matplotlib.

Techniques

This project was made using pandas and pandasql to aggregate and manipulate data, shapely to take the longitude and latitude of a crash and combine it with the speed limit csv to find the speed limit of the crash, and matplotlib and seaborn to visually graph the various data together. Various data sets were joined together and algorithms were made to be able to properly aggregate the turnstile entry data and check if the lat/long of a crash was within a certain distance of a street's speed limit or if it belonged in a slow zone.

Data

Post Pandemic Collisions in New York City

This graph plots the crashes per borough in NYC since the start of the pandemic. It seems as though crashes are slowly ticking upwards over time. The graph was made by using pandas to count and aggregate the collision data per borough per month while matplotlib was used to create the visualization.

Comparing Percent Change in Crashes in New York City

This graph plots crash data over time while also comparing speed limits per year. 20mph, 25mph, and every speed limit is compared in percentage change over time. While there is an obvious jump each year, it seems as though 20mph zones have actually not been decreasing as fast as other speed limits. This data was made by joining speed limit data and slow zone data with collision data using shapely to plot the coordinates.

Comparing Subway Entries with Crashes

This scatter plot graphs the crash data with subway entries per month for 2018 and 2019. It seems as though there is a slight positive correlation between the two. However, this can be due to various factors such as pedestrian traffic and the possibility that safer drivers are more likely to take the subway to avoid dangerous traffic. This graph was made by aggregating turnstile entry data into a monthly number and joining it with monthly crash data for the two years.
